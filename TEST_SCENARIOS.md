# תרחישי בדיקה מורחבים

## קבוצה א': פקודות בסיסיות

1. "מה כתובת ה-IP של המחשב שלי"
   - צפוי: `ipconfig`

2. "הצג את תוכן התיקייה הנוכחית"
   - צפוי: `dir`

3. "נקה את המסך"
   - צפוי: `cls`

4. "מה הזמן והתאריך של המערכת"
   - צפוי: `date /t` או `time /t`

5. "הצג את גרסת המערכת"
   - צפוי: `ver` או `winver`

## קבוצה ב': פעולות על קבצים

6. "אני רוצה למחוק את כל הקבצים עם סיומת .tmp בתיקייה downloads"
   - צפוי: `del downloads\*.tmp`

7. "צור תיקייה חדשה בשם test"
   - צפוי: `mkdir test` או `md test`

8. "העתק את הקובץ file.txt מהתיקייה הנוכחית ל-backup"
   - צפוי: `copy file.txt backup\`

9. "מחק את התיקייה old_files עם כל התוכן שלה"
   - צפוי: `rmdir /s old_files` או `rd /s old_files`

10. "שנה שם של הקובץ old.txt ל-new.txt"
    - צפוי: `ren old.txt new.txt`

## קבוצה ג': סידור ומיון

11. "לסדר את רשימת הקבצים לפי גודל מהגדול לקטן"
    - צפוי: `dir /o-s`

12. "הצג רק תיקיות בתיקייה הנוכחית"
    - צפוי: `dir /ad`

13. "הצג קבצים נסתרים"
    - צפוי: `dir /ah`

## קבוצה ד': תהליכים ומערכת

14. "איזה תהליכים רצים כרגע במערכת"
    - צפוי: `tasklist`

15. "סגור תהליך בשם notepad.exe"
    - צפוי: `taskkill /IM notepad.exe` או `taskkill /F /IM notepad.exe`

16. "הצג מידע על השימוש בזיכרון"
    - צפוי: `tasklist` או `systeminfo`

17. "הצג את כל השירותים הפועלים"
    - צפוי: `sc query` או `net start`

## קבוצה ה': חיפוש

18. "מצא קבצים בשם *.pdf בתיקייה הנוכחית"
    - צפוי: `dir *.pdf`

19. "חפש את הקובץ config.txt בכל הדיסק C"
    - צפוי: `dir C:\config.txt /s`

## קבוצה ו': רשת

20. "בדוק אם אני מחובר לגוגל"
    - צפוי: `ping google.com`

21. "הצג את החיבורים הפעילים ברשת"
    - צפוי: `netstat`

22. "הצג את הרשתות האלחוטיות הזמינות"
    - צפוי: `netsh wlan show networks`
