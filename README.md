# 🤖 CLI Agent - ממיר הוראות לפקודות Windows

כלי שממיר הוראות בשפה טבעית (עברית או אנגלית) לפקודות CLI של Windows.

## איך משתמשים?

1. הכנס הוראה בשפה טבעית — למשל: *"מה כתובת ה-IP שלי"*
2. לחץ על **תרגם לפקודה**
3. קבל את פקודת ה-CLI המתאימה — למשל: `ipconfig`

## דוגמאות

| הוראה | פקודה |
|-------|-------|
| מה כתובת ה-IP שלי | `ipconfig` |
| הצג את כל הקבצים בתיקייה | `dir` |
| איזה תהליכים רצים במערכת | `tasklist` |
| בדוק חיבור לאינטרנט | `ping google.com` |
| צור תיקייה בשם test | `mkdir test` |
| נקה את המסך | `cls` |
